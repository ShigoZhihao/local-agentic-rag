# Level 8 — MCP Integration

**Core loop:** `user input → Multi-Agent Graph → [MCP Tools: filesystem, web, DB] → output`

Extend the multi-agent system with Model Context Protocol (MCP) tools,
enabling the agents to interact with external services dynamically.

**Framework:** LangGraph + MCP

## What you learn here

- Model Context Protocol (MCP): tool servers, dynamic tool discovery
- Binding MCP tools to LangGraph agent nodes
- Filesystem, web search, and database MCP servers
- Tool authentication and access control
