# Level 11 — Composable Skills

**Core loop:** `user input → Agent → [SKILL.md lookup] → skill execution → output`

Define reusable agent capabilities as composable skills. Each skill is a
self-contained unit with its own prompt, tools, and validation logic.

**Framework:** LangGraph + SKILL.md

## What you learn here

- SKILL.md composable skill definitions
- Dynamic skill loading and registration
- Skill chaining: combining multiple skills in a single workflow
- Skill versioning and dependency management
